Weekly Questions

I tried adding two vectors in C but my code wouldn't compile. This is because vectors or arrays in C are represented as a list of numbers, the C compiler doesn't understand any vector math operations. To do vector operations there are functions in the SPHERES documentation, see: ZR SPEHRES API

Our team is having trouble submitting. We recomend running through the following checklist when submitting:

  • Check your code is written in a project that is for the competition you are submitting in
  • Make sure your team is selected
  • Make sure the correct competition that you are submitting for is selected
  • Make sure that the checklist adjacent to 'The code above is my team's official competition program' is ticked

If you still have trouble please contact us at zero-robotics.admin@sydney.edu.au
